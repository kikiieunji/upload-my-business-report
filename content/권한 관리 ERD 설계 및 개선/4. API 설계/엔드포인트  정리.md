### API 명세

##### 1. **조회 가능한 권한 조회**
- **Method**: `GET`
- **URI**: `/group-function/list`
- **설명**: 로그인한 사용자가 접근 가능한 GroupFunction 목록을 반환합니다.
- **Request**: 없음 (Query Param 등 없음)
- **Response**: `BaseResponse<List<GroupFunctionDTO>>`
    
    {
    
      "code": 200,
    
      "message": "조회 성공",
    
      "data": [
    
        {
    
          "groupFunctionId": 1,
    
          "companyGroupId": 10,
    
          "companyGroupName": "경영지원그룹",
    
          "code": "ACCIDENT_REPORT",
    
          "name": "사고 보고서",
    
          "canRead": true,
    
          "canWrite": false,
    
          "description": "사고 보고서 조회 권한"
    
        }
    
      ]
    
    }
##### 2. 전자 결재 양식 생성
- **Method**: `POST`
- **URI**: `/approval/template/create`
- **설명**: 새로운 전자결재 양식을 생성합니다. `groupFunctionIdList`를 통해 열람 권한을 설정합니다.
- **Request Body**: `ApprovalTemplateRequestDTO`
    
    {
    
      "templateName": "연차 계획서",
    
      "description": "연차 신청을 위한 양식",
    
      "groupFunctionIdList": [1, 2], // 열람 가능 권한 ID 목록
    
      "defaultApprovalLine": [ ... ], // 결재자 목록 (ApprovalTemplateLineDTO)
    
      "defaultReferenceList": [ ... ], // 참조자 목록
    
      "defaultAgreementList": [ ... ], // 합의자 목록
    
      "sections": [
    
        {
    
          "sectionTitle": "신청 정보",
    
          "sectionType": "NORMAL", // NORMAL 또는 DATA
    
          "sectionOrder": 1,
    
          "fields": [
    
            {
    
              "fieldType": "TEXT",
    
              "fieldLabel": "제목",
    
              "isRequired": true,
    
              ...
    
            }
    
          ]
    
        }
    
      ]
    
    }
    
- **Response**: `BaseResponse<Void>`

##### 3. 전자결재 양식 목록 조회
- **Method**: `GET`
- **URI**: `/approval/template/read/details/{templateId}`
- **설명**: 특정 전자결재 양식의 상세 정보를 조회합니다. `GroupFunction`, `UdpGroupFunction`을 통해 사용자의 부서/직급 정보를 매핑하고 권한을 검증합니다.
- **Response**: `BaseResponse<ApprovalTemplateDetailDTO>`
    
    {
    
      "code": 200,
    
      "message": "전자결재 양식 상세 조회 성공",
    
      "data": {
    
        "id": 1,
    
        "templateName": "연차 계획서",
    
        "authorizedFuntionList": [ ... ], // 설정된 열람 권한 (GroupFunctionDTO)
    
        "approvalLineList": [ ... ], // 결재 라인
    
        "approvalTemplateSectionList": [ ... ] // 양식 섹션 및 필드 구성
    
      }
    }
